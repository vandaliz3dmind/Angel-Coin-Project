# AngelCoin Roadmap (Testnet → Mainnet)

## Phase 1 — Prototype (Current)
- Deploy AngelCoin + registry + rewards contracts on testnet
- Test mission minting and reward flows
- Validate PoW-transition governance snapshot logic
- Build oracle mission validator

## Phase 2 — Community Testing
- Open public testnet missions
- Expand oracle ruleset
- Introduce UI for mission submission
- Conduct small bug bounty program (testnet only)

## Phase 3 — Audit Preparation
- Formal verification on core contracts
- Gas optimization pass
- Security review of oracle system and vault logic

## Phase 4 — Mainnet Candidate
- Deploy audited version of AngelCoin to mainnet
- Run snapshot → generate PoW genesis
- Launch AngelChain (PoW)
- Bridge ANGEL balances from snapshot

## Phase 5 — Ecosystem Growth
- Developer grants
- Ambassador impact missions
- Mobile wallet integration
- DAO formation for future upgrades
